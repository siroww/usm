Лабораторная работа
Учёт личных финансов
1. Цель работы
Ознакомиться с основами взаимодействия JavaScript с DOM-деревом на примере веб-приложения для учёта личных финансов.
2. Структура проекта
Проект разбит на модули:
index.html — главная страница с формой и таблицей
style.css — стили
src/index.js — точка входа, обработка событий
src/transactions.js — работа с массивом транзакций
src/ui.js — отрисовка DOM-элементов
src/utils.js — вспомогательные функции
3. Описание реализации
3.1 Транзакция
Каждая транзакция — объект вида:
{ id, date, amount, category, description }
Все транзакции хранятся в массиве transactions (transactions.js).
3.2 Добавление транзакции
При отправке формы выполняется валидация полей. Если данные корректны, создаётся объект транзакции, добавляется в массив и в таблицу. Строка окрашивается в зелёный (доход) или красный (расход) цвет.
3.3 Удаление транзакции
На таблице задан один обработчик события click (делегирование событий). При нажатии на кнопку "Удалить" транзакция удаляется из массива и из DOM.
3.4 Подсчёт баланса
После каждого добавления или удаления вызывается calculateTotal(), которая суммирует amount всех транзакций и обновляет элемент #total.
3.5 Просмотр деталей
При клике на строку таблицы (не на кнопку удаления) внизу страницы отображается полное описание транзакции.
4. Ответы на контрольные вопросы
1. Как получить доступ к элементу на странице?
Используются методы: document.getElementById(), document.querySelector(), document.querySelectorAll(). Например:
const el = document.getElementById("total");
2. Делегирование событий
Делегирование событий — приём, при котором обработчик назначается на родительский элемент, а не на каждый дочерний. Событие "всплывает" от дочернего к родителю. В работе один обработчик click назначен на <table> и обрабатывает клики по кнопкам удаления и строкам через e.target.
3. Как изменить содержимое элемента DOM?
Через свойства textContent (текст) или innerHTML (HTML). Например:
el.textContent = "Новый текст";
4. Как добавить новый элемент в DOM?
Создать элемент через document.createElement(), заполнить его и добавить через appendChild() или insertAdjacentElement(). Пример:
const tr = document.createElement("tr");
tbody.appendChild(tr);
5. Вывод
В ходе лабораторной работы было разработано модульное веб-приложение для учёта личных финансов. Были применены: разбивка кода на модули ES Modules, работа с DOM (создание, удаление элементов), делегирование событий, валидация формы и JSDoc-документирование.
