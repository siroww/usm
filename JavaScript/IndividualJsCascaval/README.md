# CryptoBoard

Веб-приложение для отслеживания курсов криптовалют в реальном времени.

## Автор

Индивидуальный проект по дисциплине «JavaScript»

## Описание

CryptoBoard — интерактивный дашборд криптовалют, построенный на чистом Vanilla JS. Приложение загружает данные с публичного API CoinGecko и отображает топ-10 монет по рыночной капитализации.

## Основные функции

- Отображение топ-10 криптовалют с ценой и изменением за 24ч
- Поиск монет по названию или символу
- Сортировка по рыночной капитализации или цене
- Переключение валюты: USD, EUR, RUB
- Добавление монет в избранное (сохраняется в localStorage)
- Модальное окно с детальной информацией по монете
- Адаптивный интерфейс для мобильных устройств

## Структура проекта

```
cryptoboard/
├── index.html   — разметка страницы
├── style.css    — стили
├── api.js       — запросы к CoinGecko API
├── storage.js   — работа с localStorage (избранное)
├── format.js    — форматирование цен и процентов
├── render.js    — отрисовка карточек и модального окна
├── app.js       — состояние приложения и обработчики событий
└── README.md
```

## Запуск

Проект не требует установки зависимостей и сборщиков.

1. Скачайте или клонируйте репозиторий
2. Откройте файл `index.html` в браузере

> Для корректной работы требуется подключение к интернету (загрузка данных с CoinGecko API).

## Использованные технологии

- Vanilla JavaScript (ES6+)
- CSS3 (Grid, Flexbox, анимации)
- [CoinGecko API](https://www.coingecko.com/en/api) — бесплатный публичный API криптовалют
- Google Fonts (DM Sans, Space Mono)
