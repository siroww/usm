Отчет о функциях массива

1. forEach:
   Функция forEach перебирает каждый элемент массива и вызывает для него функцию обратного вызова. Не возвращает новый массив, используется для выполнения действий над элементами.

2. map:
   Функция map создает новый массив, применяя функцию обратного вызова к каждому элементу исходного массива. Возвращает новый массив с результатами.

3. filter:
   Функция filter создает новый массив, содержащий только те элементы, для которых функция обратного вызова вернула true. Используется для фильтрации массива.

4. find:
   Функция find возвращает первый элемент массива, который удовлетворяет условию в функции обратного вызова. Если такого элемента нет, возвращает undefined.

5. some:
   Функция some проверяет, есть ли хотя бы один элемент в массиве, удовлетворяющий условию. Возвращает true или false.

6. every:
   Функция every проверяет, удовлетворяют ли ВСЕ элементы массива условию. Возвращает true, если все элементы проходят проверку, иначе false.

7. reduce:
   Функция reduce накапливает результат, применяя функцию обратного вызова к каждому элементу массива с аккумулятором. Возвращает итоговое значение аккумулятора.
