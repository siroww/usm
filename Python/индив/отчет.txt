 

Отчёт по Python/индив

1. Назначение проекта:
Проект предназначен для учета прочитанных, читаемых и желаемых книг.
Приложение помогает вести личную библиотеку и записывать заметки о книгах.

2. Структура проекта:
- app.py — основной Flask-файл.
- requirements.txt — зависимости Python.
- database.db — SQLite-база данных с таблицей books.
- templates/ — HTML-шаблоны для страниц.
- static/ — статические ресурсы для внешнего вида.

3. Содержание приложения:
- Главная страница с краткой информацией.
- Страница списка книг с фильтрацией и поиском.
- Страница детальной информации о выбранной книге.
- Форма добавления книги с полями статуса и заметок.
- Редактирование данных книги через форму.
- Удаление книги с подтверждением.

4. Модель данных:
Таблица books содержит поля:
- id — уникальный идентификатор.
- title — название книги.
- author — автор.
- genre — жанр.
- pages — количество страниц.
- rating — оценка книги.
- created_at — дата добавления.
- status — статус чтения.
- notes — заметки пользователя.

5. Особенности реализации:
- База данных создается автоматически при запуске.
- В базу добавляются примеры книг при первом запуске.
- Сортировка списка книг поддерживает несколько полей.
- Добавлен поиск по названию и автору.
- Поддержка фильтрации по жанру и статусу.
- Статусы: Прочитано, Читаю, Хочу прочитать.
- Применяется валидация числовых и текстовых полей.

6. Маршруты приложения:
- / — главная страница.
- /books — список книг.
- /books/<int:book_id> — страница книги.
- /add — добавление книги.
- /books/<int:book_id>/edit — редактирование книги.
- /books/<int:book_id>/delete — удаление книги.

7. Функции валидации:
- Проверка заполненности названия, автора и жанра.
- Убедиться, что количество страниц является целым и больше нуля.
- Проверка рейтинга как числа от 0 до 10.
- Проверка корректного статуса книги.
- Сохранение заметок без обязательного ввода.

8. Пользовательский опыт:
- Понятное управление книгами через веб-интерфейс.
- Быстрый поиск и сортировка.
- Возможность записывать личные заметки.
- Статусы помогают отслеживать прогресс чтения.
- Кнопка удаления удаляет выбранную книгу.

9. Запуск проекта:
1. Установить виртуальное окружение Python.
2. Установить зависимости из requirements.txt.
3. Запустить app.py.
4. Перейти в браузере по адресу http://127.0.0.1:5000.

10. Рекомендации:
- Провести тестирование форм в браузере.
- Проверить работу фильтрации и поиска.
- Переписать шаблоны для лучшего визуального оформления.
- Добавить страницу с общей статистикой.

11. Вывод:
Проект готов к использованию как простой личный трекер книг.
Отчёт содержит все главные аспекты приложения в краткой форме.
